Dandre Vermaak

## logging in
use the admin email `admin@epiuseapp.com` and the password `EpiUse123!` to get in and see everything.

## the main registry
- this is the big list of all staff. 
- you can search for a name or role in the top bar.
- if you are an admin you can see the salary and delete buttons. 
- there is a download icon at the top right to get the whole list as a CSV.

## enrollment (adding people)
- go to the Add Employee tab to put someone new in the system.
- make sure you pick their manager from the dropdown so they show up in the right place on the chart.
- if they are the boss (CEO), just leave the manager empty.

## org chart
- click the Hierarchy View to see the tree map.
- use your trackpad or mouse to zoom and move the map around. 
- click a person's box to see their full details in the side panel on the right.
- you can search here too and it will jump right to the person you found.

## security
- you can change your own password in the Change Password tab. 
- admins can reset other people's passwords from the registry list if they forget it.
